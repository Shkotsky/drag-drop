# Drag-Drop  :dizzy:

**Dragging** and **dropping** elements, in this case letters into words.

Works with native drag,  **mouse**  and **touch events**.

**Desktop** and **mobile** friendly.

## Build with:

- **HTML**
- **CSS**
- **JavaScript**
- **Web APIs** ( HTML Drag and Drop API and Touch events )
